# QA Automation Framework – Doon Architects Website

## Project Overview

This project demonstrates end-to-end automation testing of a production-level architecture website (doonarchitects.com). It covers UI automation, API testing, and real-world user workflows.

## ⚙️ Tech Stack

- Python
- Selenium WebDriver
- Pytest
- Requests (API testing)

---

## 🚀 Features Tested

### UI Automation

- Navigation across multiple pages
- Contact form submission
- Careers/job application workflow
- File upload functionality

### Functional Testing

- Form validation
- Dropdown selection
- Required field validation

### Negative Testing

- Empty form submission
- Invalid input handling

### API Testing

- Contact form API validation
- Status code verification

---

## 🧱 Framework Design

- Page Object Model (POM)
- Modular structure (pages, tests, data)
- Reusable components

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
pytest
```
